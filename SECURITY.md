# Security Policy

## Supported Versions

Security updates are currently provided for the latest version on the `main` branch.

## Reporting a Vulnerability

Please do not disclose security vulnerabilities in public issues.

Report vulnerabilities privately with:

- A clear description of the issue
- Impact assessment
- Reproduction steps or proof of concept
- Suggested remediation (if available)

Maintainers will acknowledge reports and work toward a fix and coordinated disclosure.

## Data Handling Notes

- This app processes local Instagram export files.
- Do not commit personal export data to the repository.
- Review files before sharing screenshots or logs.
