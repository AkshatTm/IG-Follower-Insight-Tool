# Code of Conduct

## Our Commitment

This project is committed to providing a welcoming, respectful, and harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Give and receive feedback professionally.
- Focus discussions on technical and project outcomes.

## Unacceptable Behavior

- Harassment, discrimination, or personal attacks
- Trolling, insulting, or demeaning comments
- Sharing private information without consent

## Enforcement

Project maintainers are responsible for clarifying and enforcing community standards. They may remove, edit, or reject comments, commits, issues, and other contributions that violate this code.

## Reporting

If you experience or witness unacceptable behavior, contact the maintainer privately through repository contact channels.

## Scope

This code of conduct applies in all repository spaces, including issues, pull requests, and discussions.
