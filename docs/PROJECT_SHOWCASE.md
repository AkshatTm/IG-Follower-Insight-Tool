# Project Showcase: Instagram Red Flags

## Elevator Pitch

Instagram Red Flags is a desktop-first analytics utility that helps users identify accounts that do not follow them back, preserve influencer and VIP relationships, and export practical unfollow lists in seconds.

## Why This Project Exists

Manually comparing followers and following lists from Instagram exports is error-prone and time-consuming. This project converts a tedious process into a guided, local, privacy-aware workflow.

## What Was Built

- End-to-end 4-screen desktop experience
- Robust parser for varying Instagram JSON structures
- Deep Scan filtering workflow with persistent whitelisting
- Export tools for TXT, CSV, and clipboard handoff
