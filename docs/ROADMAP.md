# Product Roadmap

## Vision

Make Instagram relationship auditing local, clear, and action-ready with a polished desktop user experience.

## Phase 1: Foundation (Completed)

- Multi-screen desktop flow
- JSON upload and parsing
- Non-follower quick analysis
- Smart filtering and whitelist persistence
- CSV and clipboard export tools

## Phase 2: Reliability and Distribution

- Add automated unit tests and CI checks
- Improve edge-case messaging and recovery UX
- Package for Windows distribution (single executable)
- Add release tagging and changelog discipline

## Phase 3: Product Experience Enhancements

- Optional import history and session snapshots
- Better analytics cards (percentages and trends)
- Enhanced search/filter controls (batch operations)
- Export customization (include metadata columns)

## Phase 4: Advanced Insights (Optional)

- Plugin model for additional social-graph analyses
- Heuristic scoring for relationship quality signals
- Privacy-preserving local intelligence workflows

## Success Metrics

- Setup time under 5 minutes
- Analysis completion under 30 seconds for normal export sizes
- Zero crash tolerance on malformed inputs
- Repeat usability from whitelist persistence without rework
